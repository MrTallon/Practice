# 设计模式


> 1.设计模式是在特定环境下解决特定问题的处理办法  
2.可以把设计模式理解成生活中的套路或者围棋中的定式  
3.设计模式是前人经验的总结，是一种发现的过程

> 开闭原则：对外扩展开放，对内修改封闭
